  • •

    UX design

    UX design signifie User Experience Design. En Français, il s’agit du design de l’expérience utilisateur.

  • •

    UI design

    Le terme UI est l’abréviation d’user interface qui désigne l’interface utilisateur. Le rôle de l’UI designer consiste à concevoir une interface agréable par le biais duquel l’homme entre en contact avec le produit.

  • •

    SEO

    Le SEO (Search Engine Optimization) est l’art d’optimiser un projet web et son environnement pour les moteurs de recherche (Google, Bing, Yahoo…).
    Plus communément appelé référencement naturel, but est d’augmenter la qualité, la compréhension, mais surtout la visibilité d’un site internet au sein des résultats des moteurs de recherche.
